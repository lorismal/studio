# **App Name**: Startup Compass

## Core Features:

- Idea Definition: Define the startup idea and categorize it using an AI tool to assist in identification and placement.
- Process Definition: Define a main start-up management process using established methodologies (Lean Startup, SMART, OKR, Innovation Accounting) displayed in a navigation on the left of the screen.
- Objective Creation: Create objectives based on the chosen startup management model.
- Objective Selection: Enable users to select applicable objectives, with some pre-selected based on the user's start-up type.
- Note-Taking Section: Provide a dedicated space for taking notes within the dashboard, ensuring all information is readily accessible and well-organized.
- Markdown Editor Integration: Integrate a Markdown editor that allows users to create rich text notes, format content effectively, and enhance the overall note-taking experience.
- Process Attachment: Allow users to attach notes created in the Markdown editor directly to specific processes or steps within their startup management workflow.
- Notebook Access: Implement a comprehensive notebook feature that stores all recorded notes in an organized manner, facilitating easy access and review.
- Manual Tagging: Implement a system for manual tagging of notes using a predefined list of tags based on the type of startup.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light blue-gray (#ECEFF1), for a professional, understated feel.
- Accent color: Grey (#757575) for call-to-action elements and important highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern reading experience.
- Code font: 'Source Code Pro' for displaying code snippets within notes.
- Use simple, clear, and professional icons to represent different startup stages and methodologies.
- Implement a clear, intuitive dashboard layout with a sidebar for process selection and a main content area for objectives and notes.
- Fast and standard animations to transition between different sections and highlight updates or new notes, maintaining a minimalist approach.
